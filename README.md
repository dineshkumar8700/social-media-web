# Web Social Media

## Overview

Web Social Media is a Twitter-like social media web application that allows
users to share posts, interact with others, and build connections.

Users can access their dashboard after successful authentication. From the
dashboard, they can create posts, view posts from others, like them, and leave
comments.

---

## Objectives

- Build a real-time, interactive social media web application
- Understand state management between the client and server
- Gain confidence in building full-stack applications
- Learn system design principles for scalable systems

---

## Core Features

- User authentication (login / signup)
- Personal dashboard
- Create and view posts
- Like and comment on posts
- Interactive user feed

---

## Future Plans
